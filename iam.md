## AWS IAM

IAM - Identity Access Management

IAM has following elements;

1. **Policies**: Define permissions to control user and service access to AWS resources.
1. **Users**: Individual accounts with specific permissions to access AWS resources.
1. **User Groups**: Collection of users with common permissions applied through shared policies.
1. **Roles**: AWS identities with specific permissions, assumed by trusted entities (users, services) to access resources without using long-term credentials.
